React Reddit Project


This is a Reddit style application that allows users to make posts about topics of their choosing. Users can also create new categories and leave comments about existing posts. This is a full MERN Stack application.

Texas Beer Wars

Texas Beer Wars is a full stack interactive web application. This app is designed to allow users to vote on their favorite beer in a battle to find the best native Texan beer. This project was built using an MVC structure. The backend of this application incorporates Node.js and Express to route both get and post requests. Mocha and Chai are used as a framework to test the Node.JS RESTful functionality. The app data is stored in MySQL and JawsDB using a Sequelize ORM.

NYT React Search

This is a full stack application that utilizes the complete MERN Stack. AJAX requests through Axios are used to query the NYT API for news articles related to the search term within the specified beginning and ending years. The React components are then rendered to display the search results and the articles saved by the user for later viewing.

Cheerio News Scraper

This application is a full stack website that utilizes Node.js, Express, MongoDB, Mongoose, Handlebars.js and Cheerio. Node.js and Express are used to query and route data into the app and display the appropriate pages. Cheerio is used to scrape the news data from Rawstory, and the data is stored in MongoDB using Mongoose. The front-end HTML is displayed using Handlebars.js. Users can both leave and delete comments for each article.

Friend Finder

Friend Finder is similar to a dating app. This is a full stack Restful API website which sets the HTML and API routes for posting and requesting data through Express and Node.JS.

Giphy API

Giphy API is a front end application that utilizes JavaScript, jQuery and AJAX to query GIPHY API to search for GIFs. GIPHY API is a website with a large library of GIFs that can be queried through a REST-like interface accessible through any HTTP enabled programming language.

Hangman Game

Hangman is a rendition of the classic game hangman with a movie title theme. This application was created with Vanilla JS and is not mobile responsive. This game must be played on a system that includes a keyboard.

Sequelized Burger Logger

This application is a full stack MVC "burger logger" that utilizes MySQL, Sequelize, Node.JS, Express and Handlebars.JS. Node.JS, MySQL, and Sequelize are used to query and route data into the app, and Handlebars.JS is used to generate the HTML.
