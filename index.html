Texas Beer Wars


Overview: Texas Beer Wars is a full stack interactive web application. This app is designed to allow users to vote on their favorite beer in a battle to find the best native Texan beer. This project was built using an MVC structure. The backend of this application incorporates Node.js and Express to route both get and post requests. Mocha and Chai are used as a framework to test the Node.JS RESTful functionality. The app data is stored in MySQL and JawsDB using a Sequelize ORM.
